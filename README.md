# 🤖 CodSoft Artificial Intelligence Internship

This repository contains the projects I completed as part of the **Artificial Intelligence Internship** at **CodSoft**. Each project was developed using Python to strengthen my understanding of AI fundamentals, problem-solving, and interactive application development.

## 📂 Projects Included

### 1. 🤖 SoftChat – Rule-Based Chatbot

A simple chatbot that interacts with users using predefined rules and responses.

#### Features
- Interactive command-line chatbot
- Responds to common greetings and questions
- Rule-based conversation flow
- Easy-to-use interface
- Built using Python

---

### 2. 🎮 AI Tic-Tac-Toe

An AI-powered Tic-Tac-Toe game where players compete against a computer opponent.

#### Features
- Two difficulty levels:
  - **Easy Mode** – AI makes random moves.
  - **Hard Mode** – AI plays strategically by blocking the player's winning moves and making intelligent decisions.
- Player vs AI gameplay
- Win, lose, and draw detection
- Interactive command-line interface

---

### 3. 🎬 SmartFlix – Movie Recommendation System

A genre-based movie recommendation system that suggests movies according to the user's preferences.

#### Features
- Supports multiple genres:
  - Action
  - Comedy
  - Horror
  - Science Fiction
  - Romance
  - Animation
  - Thriller
  - Fantasy
- Interactive command-line interface
- Curated movie recommendations
- Simple and user-friendly design

---
